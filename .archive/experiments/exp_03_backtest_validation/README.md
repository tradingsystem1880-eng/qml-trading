# Experiment 03: Backtest Validation

## Purpose
Validation and revalidation of detected patterns against historical trades.

## Files
- `revalidation_equity.csv` - Equity curve data
- `revalidation_summary.csv` - Backtest summary statistics
- `revalidation_trades.csv` - Individual trade records
- `pattern_audit_matches.csv` - Matched patterns
- `pattern_audit_unmatched.csv` - Unmatched patterns requiring investigation
