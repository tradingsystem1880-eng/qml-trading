Methodology

Pattern Detection Algorithm

The QML (Quasimodo) pattern detection system uses a hierarchical swing detection algorithm to identify five-point reversal patterns in price action.

Detection Layers

  1. Geometric Layer: Identifies swing highs/lows using ATR-based thresholds
  2. Significance Layer: Filters swings by magnitude and market context
  3. Pattern Layer: Validates 5-point QML structure (P1→P2→P3→P4→P5)

Scoring Components

  • Head Extension Score (22%)
  • Break of Structure Efficiency (18%)
  • Shoulder Symmetry (12%)
  • Swing Significance (8%)
  • Volume Spike (10%)
  • Path Efficiency (10%)
  • Trend Strength (10%)
  • Regime Suitability (10%)

Regime Definitions

  • RANGING: Low ADX (<20), price oscillating in range - Ideal for QML patterns
  • TRENDING: High ADX (>30), sustained directional move - Lower pattern success
  • VOLATILE: High ATR relative to recent history - Wider stops needed
  • EXTREME: ADX >35 with high volatility - Avoid trading
